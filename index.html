Chai
Tailwind

A lightweight utility-first CSS engine. Write chai-* classes in your HTML — the engine scans, parses, and applies inline styles automatically.

<div class="chai-p-4 chai-bg-blue chai-text-white chai-rounded-lg">Hello Chai</div>
▶ Try Playground View Reference

Approach

ChaiTailwind intercepts classes on page load and converts them to inline styles using a pattern parser.

01

DOM Traversal

On DOMContentLoaded, the engine queries all elements and filters those with chai-* prefixed classes.

02

Pattern Parsing

Each class is matched against regex patterns. Values are extracted and mapped to CSS properties using lookup tables.

03

Inline Style Injection

Parsed styles are applied via element.style, converting kebab-case to camelCase automatically.

04

Class Cleanup

Original chai-* classes are removed from the DOM, leaving clean, unstyled markup.

All Utilities

A full reference of supported chai-* class patterns.

Spacing
chai-p-{n} padding: {n*4}px
chai-px-{n} padding X axis
chai-py-{n} padding Y axis
chai-pt/pr/pb/pl-{n} padding per side
chai-m-{n} margin: {n*4}px
chai-mx-auto margin: 0 auto
chai-mt/mr/mb/ml-{n} margin per side
Colors
chai-bg-{color} background-color
chai-bg-{color}-{shade} e.g. chai-bg-blue-500
chai-text-{color} color
chai-border-{color} border-color
red, blue, green, yellow, orange,
purple, pink, teal, indigo, cyan…
Typography
chai-text-{size} xs → 6xl
chai-font-{weight} thin → black
chai-text-center text-align: center
chai-uppercase text-transform
chai-leading-{val} line-height
chai-tracking-{val} letter-spacing
chai-truncate text overflow ellipsis
Layout & Flex
chai-flex display: flex
chai-flex-col flex-direction: column
chai-items-center align-items: center
chai-justify-between justify-content
chai-gap-{n} gap: {n*4}px
chai-grid-cols-{n} grid columns
chai-hidden display: none
Borders & Radius
chai-border 1px solid border
chai-border-{n} border-width: {n}px
chai-border-dashed border-style: dashed
chai-rounded border-radius: 0.25rem
chai-rounded-lg border-radius: 0.5rem
chai-rounded-full border-radius: 9999px
chai-shadow-{sz} sm / md / lg / xl / 2xl
Sizing & Position
chai-w-{n} width: {n*4}px
chai-w-full width: 100%
chai-h-{n} height: {n*4}px
chai-relative position: relative
chai-absolute position: absolute
chai-top-{n} top: {n*4}px
chai-opacity-{n} opacity: 0–100
